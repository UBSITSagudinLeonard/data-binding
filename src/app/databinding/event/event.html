C — Event Handling

Mastering the flow of data from the User Interface back to the Component Logic using Angular's Event Binding syntax.

Core Concept

Event Binding allows your application to capture and respond to user interactions—such as clicks, keystrokes, and form submissions. It establishes a one-way data flow from the View to the Component.

Syntax: (targetEvent)="statement"

The targetEvent identifies the specific DOM event, while the statement executes the corresponding TypeScript logic defined in your controller.
